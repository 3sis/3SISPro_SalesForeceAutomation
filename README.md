<h2 style="color:cyan">Project Installation</h2>
   
```bash
     git clone https://github.com/3sis/3SISPro_SalesForeceAutomation.git
     cd 3SISPro_SalesForeceAutomation
     composer install
     cp .env.example .env

     Set up .env file

     php artisan key:generate

     php artisan serve
 ```
